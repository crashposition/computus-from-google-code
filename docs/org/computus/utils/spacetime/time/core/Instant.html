Classpublic class Instant
InheritanceInstant flash.events.EventDispatcher

An Instant represents a single point in time. It contains only a temporal value and has no geographical component. An Instant() can be used on it's own or extended with the addition of specific calendarical rules. Changing the value of an Instant will broadcast a CHANGE event.

Property detail
timeproperty
time:Number  [read-write]

The total number of millieconds before or after the standard epoch (1.1.1970). This property is polymorphic with the AS3 Date class. Changing the value of time will broadcast a Event.CHANGE event.

Instant()constructor
public function Instant(... args)

Creates a new instance of the Instant() class.

The follow parameters are accepted by the constructor:

Instant() sets time to the current date and time.
Instant( ms:Number ) sets time to the value passed in the constructor.
Instant( d:Date ) sets time to the value of the AS3 Date passed in the constructor.
Instant( d:Instant ) sets time to the value of the Instant passed in the constructor.
